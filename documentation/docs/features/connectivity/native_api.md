# Native Api

This uses the same API as ESPHome.

## Basic Configuration

```yaml
# This make the native api available
api:
```

```yaml
api:
  # Optional: Default Port is 6053
  port: 6053
  encryption_key: 'your_secure_key_here' # Generate
```

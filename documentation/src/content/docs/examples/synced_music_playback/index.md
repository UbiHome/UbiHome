---
title: 'Synchronized music playback'
description: 'Demonstrate how to play music in sync across multiple devices.'
---

Thanks to [Music Assistant](https://github.com/music-assistant) and the [Sendspin](https://www.sendspin-audio.com/) integration, UbiHome can be used to play music in sync across multiple devices. This allows you to create a multi-room audio setup with UbiHome as the client on each device.

1. Install and set up [Music Assistant](https://www.music-assistant.io/installation/).
2. Enable Sendspin on all your UbiHome devices by adding the following to your configuration:

```yaml title="config.yml"
sendspin:
```
> For more advanced configuration options check the [Sendspin Platform](/features/platforms/sendspin/).

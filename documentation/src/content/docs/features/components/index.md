# Components

Components make data or functionality available for external systems via [Connectivity Features](/features/connectivity/), or for reuse in other components.

[Platforms](/features/platforms/) provide the implementation for component behavior.

A component entry is defined under a component key (for example `sensor:` or `switch:`), and each entry selects a `platform`.

## Common Component Attributes

All components share a set of [common attributes](/features/components/base/) that can be used to customize their behavior.
